Diseño de 2 Columnas

Este tema trata sobre las hojas de estilo en cascada o CSS (cascading Style Sheets). Aquí vemos los conceptos básicos, la resolución de conflictos de reglas, el modelo de caja y las técnicas de diseño CSS. Luego se ven consultas de medios y sitios web que responden al código sin ningún marco, pues sería un error saltar directamente a un framework receptivo como Bootstrap sin entender primero los fundamentos o bases de las CSS.

En la web, el contenido lo es todo. Ya vimos que HTML define la estructura de ese contenido. Cuando se quiere contenido sin procesar, sólo se colocan esos HTML en un servidor y listo. Pero no tendría diseño y por lo tanto, muy probablemente a nadie le gustaría o nadie lo consumiría. Es como la analogía de una casa, a todos nos gustaría vivir en una casa bien estructurada, con las tuberías tapadas y así. Es la diferencia entre sentirse protegido de elementos externos o de sentirse como en casa. El punto es que cuando se trata de un sitio web, la estructura no es suficiente. Se necesita diseñar el contenido de una manera que sea agradable y útil para el usuario. El uso del color, el posicionamiento, el tamaño, entre otras cosas, es parte de eso.
